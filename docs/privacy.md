# Политика конфиденциальности
**в отношении обработки персональных данных**

## 1. Общие положения
1.1. Настоящая Политика определяет порядок обработки информации о физических лицах (Пользователях), которая может быть получена **ИП «B.Motion»** (Казахстан) и **ИП Бородиной Анастасией Алексеевной** (РФ) (совместно — **Оператор**) при использовании сайта [borodina.online](https://borodina.online) и бота `@borodina_pilates_bot`.
1.2. Основания обработки:
*   Для граждан РК: Закон РК № 94-V «О персональных данных и их защите».
*   Для граждан РФ: Федеральный закон РФ № 152-ФЗ «О персональных данных».

## 2. Состав персональных данных
2.1. Мы собираем только данные, необходимые для оказания услуг:
*   Имя, Фамилия (из Telegram);
*   Email (для чеков);
*   Номер телефона (при необходимости);
*   Telegram User ID и username;
*   История заказов.

2.2. **Платежные данные:** Оператор **не собирает и не хранит** данные банковских карт. Все транзакции проходят через защищенные шлюзы платежных провайдеров (PCI DSS).

## 3. Цели обработки
*   Исполнение Договора оферты (доступ к курсу);
*   Направление чеков об оплате;
*   Техническая поддержка;
*   Информирование о статусе подписки и новых курсах.

## 4. Передача данных
4.1. Мы можем передавать данные только:
*   Платежным системам (для проведения оплаты);
*   Платформе Telegram (для работы бота);
*   Государственным органам (по закону).
4.2. **Трансграничная передача:** Данные могут передаваться между РФ и РК для обеспечения работы сервиса.

## 5. Сроки хранения и права
5.1. Данные хранятся в течение срока действия Договора и 5 лет после (для бухгалтерии).
5.2. Пользователь имеет право требовать уточнения или удаления своих данных, написав на [support@borodina.online](mailto:support@borodina.online).
