#ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ
в отношении обработки персональных данных

г. Алматы, Республика Казахстан
Дата публикации: 23.12.2025

Настоящая Политика конфиденциальности (далее — Политика) действует в отношении всей информации, которую ИП «Бородина Анастасия Алексеевна» (далее — Оператор) может получить о пользователе во время использования им Telegram-бота @borodina_pilates_bot.

#1. ОБЩИЕ ПОЛОЖЕНИЯ
1.1. Обработка персональных данных осуществляется в соответствии с Законом Республики Казахстан от 21 мая 2013 года № 94-V «О персональных данных и их защите».

#2. КАКИЕ ДАННЫЕ МЫ СОБИРАЕМ
2.1. Оператор может обрабатывать следующие данные Пользователя:
— Имя, Фамилия (как указано в профиле Telegram);
— Telegram User ID (уникальный идентификатор);
— Имя пользователя (@username);
— История заказов и оплат.
2.2. Оператор НЕ собирает и НЕ хранит данные банковских карт. Обработка платежных данных производится на защищенных серверах платежной системы Robokassa.

#3. ЦЕЛИ СБОРА ДАННЫХ
3.1. Предоставление доступа к обучающим материалам (идентификация пользователя).
3.2. Техническая поддержка и обратная связь.
3.3. Направление уведомлений об истечении срока подписки и новых курсах (сервисные сообщения).

#4. ПЕРЕДАЧА ДАННЫХ ТРЕТЬИМ ЛИЦАМ
4.1. Оператор не передает персональные данные третьим лицам, за исключением случаев:
— Передача необходима для обработки платежа (Платежной системе);
— Передача предусмотрена законодательством РК (по запросу госорганов).

#5. БЕЗОПАСНОСТЬ
5.1. Оператор принимает необходимые технические меры для защиты данных от несанкционированного доступа.

#6. КОНТАКТЫ
6.1. По вопросам, касающимся обработки персональных данных, Пользователь может обратиться к Оператору по адресу электронной почты: support@borodin.live.
